# Centro_Adopcion
Este sistema está diseñado para digitalizar y centralizar la administración de un Centro de Adopción de Mascotas. Su objetivo principal es sustituir los registros manuales por una base de datos eficiente que permita llevar un control histórico de cada animal y su proceso de integración en un nuevo hogar.

El sistema se divide en tres pilares fundamentales:

Gestión de Inventario Animal: Permite registrar la entrada de nuevas mascotas, detallando información crítica como especie, raza, edad aproximada y estado de salud inicial.

Control de Adoptantes: Mantiene un registro de las personas interesadas, validando su información de contacto y permitiendo dar seguimiento a sus solicitudes.

Flujo de Adopción: Vincula a una mascota con un adoptante, cambiando automáticamente el estado de disponibilidad del animal para evitar duplicidad en las adopciones.


Caracteristicas:
Registro y edición de mascotas (nombre, edad, especie, etc.).
Gestión de estados de adopción.

tecnologia utilizada:
Lenguaje:** Python
Framework Web: Flask
Base de Datos: MySQL
Frontend: HTML5, CSS3, Bootstrap
Herramientas: Tilix, Git
